# integer_stats.js
Program Assignment 1 – Integer Statistics Calculator
Overview

This JavaScript program allows a user to enter a series of integers from the keyboard. The program stores the integers in an array and calculates the following statistics:

Mean
Median
Count
Minimum Value
Maximum Value

The program includes input validation and error handling to ensure only valid integers are accepted. Users can enter q at any time to stop entering numbers and display the results.

Requirements
Node.js
Git
GitHub
Running the Program

Open a terminal in the project folder and run:

node integer_stats.js
Example Input
10
20
30
40
50
q
Example Output
Count: 5
Mean: 30
Median: 30
Minimum value: 10
Maximum value: 50
